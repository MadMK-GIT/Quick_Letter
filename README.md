# QuickLetter

QuickLetter is a simple web project built using HTML and CSS. It provides a clean and basic interface for writing or displaying letters.

## 🚀 Features

* Simple and clean design
* Easy to use layout
* Built using only HTML and CSS

## 🛠️ Technologies Used

* HTML
* CSS

## 📂 How to Use

1. Download or clone the repository
2. Open the `index.html` file in your browser

## 📌 Note

This project is created for learning purposes and demonstrates basic web design using HTML and CSS.
